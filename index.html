Project #1

Food Recipe App

The Food Recipe App is a full-stack web application developed using the MERN stack (MongoDB, Express.js, React, and Node.js). It enables users to create, view, edit, update, and delete recipes with the option to upload images for each recipe. All recipe data, including images, is securely stored in a MongoDB database, while the backend provides RESTful APIs to manage data operations efficiently. This application highlights the integration of media handling with CRUD functionality for a smooth user experience.

ReactJS ,Express.js ,MongoDB ,NodeJS

Project #3

Bank Management System

The Bank Management System is a simple application designed to handle basic banking operations. It allows users to register and login to their accounts securely. Once logged in, account holders can credit, debit, and check their account balance. All user account data is stored and managed in local storage using text files, ensuring lightweight data handling and easy access. This project demonstrates core concepts of file handling, authentication, and basic financial transactions.

Java, Swing, Oops

Project #4

Retail billing System

The Retail Billing System is a simple application designed to streamline the billing process. It first collects customer details, then allows the user to add purchased items. By clicking the “Total” button, the application calculates the total bill amount automatically. It also provides options to send the bill via email and print the bill for the customer. This project demonstrates the integration of data entry, billing calculation, and digital receipt generation for retail operations.

Python, tkinter
